Как установить?

Вы можете установить расширение для двух браузеров:
  • Установить для Chrome
  • Установить для Opera

Что дальше?

Затем нужно зайти на страницу социальной сети vk.com. Вы увидите окошко авторизации. Можно либо разрешить авторизацию через соц. сеть, либо закрыть окошко, тем самым отключив поддержку этой соц. сети. Включить ее можно будет в настройках расширения. После авторизации расширение готово к использованию. Можно увидеть новую кнопку рядом с привычной "мне нравится".

Настройки

У расширения есть настройки, попасть в них можно следующим образом:
  • Для Chrome: Tools (Инструменты) -> Extensions (Расширения) -> Youhonest -> Options (Настройки)
  • Для Opera: Tools (Инструменты) -> Extensions (Расширения) -> Manage Extensions (Управление расширениями) -> Youhonest -> значок ключа -> Preferences (Настройки)

Настройки позволяют включать/отключать расширение для конкретной социальной сети и менять язык расширения.

Например, интерфейс приложения на английском языке:

Я нашел баг, что делать?

Пожалуйста, сообщите об этом сюда.

Я хочу помочь проекту, что делать?

Напишите мне, я буду очень рад.
