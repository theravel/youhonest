Во всех социальных сетях существует кнопка Like или "Мне нравится". Youhonest - это расширение для вашего браузера, которое добавляет кнопку Dislike или "Мне не нравится". Расширение поддерживает социальную сеть Вконтакте.

На данный момент поддерживаются следующие браузеры:

  • Youhonest для Opera
  • Youhonest для Chrome

Подробное описание возможностей расширения расположено на странице описания приложения.

Разработка проекта приостановлена, т.к. по решению администрации Вконтакте приложение Youhonest нарушает правила сайта. Подробнее тут.

Все исходные коды выложены на github: https://github.com/theravel/youhonest. Сервис продолжит свое существование, однако я не буду его поддерживать (фиксить баги, добавлять новую функциональность), т.к. я не вижу в этом смысла. Любой желающий может взять исходники за основу подобного сервиса.
